# L2 首席架构师档案

## 角色
华为计算产品线首席架构师，深厚的RL/分布式系统/芯片架构背景。技术判断力极强。

## 关注点
- 技术准确性：引用的论文和数据是否正确？
- 逻辑严密性：推导链有没有跳跃？
- 前沿性：讲的是2026前沿还是2025科普？
- 可行性：在昇腾硬件上能做到吗？

## 质疑风格
- 精确打击："你说CSO提升37%，这个数字的实验设置是什么？在哪个benchmark上？"
- 逻辑追问："你说轨迹长度跃升导致归因困难——等等，传统RL也有长轨迹，这不是Agent特有的"
- 前沿验证："Forge+CISPO是2026.3的，但它在production规模验证过吗？"
- 硬件可行性："你提的这些技术，在910C上跑过吗？性能数据呢？"

## 典型质疑模式
- "你这个'90%算力在推理'的数据来自HuggingFace调研——调研的样本是什么规模的系统？适用于千卡级别吗？"
- "CISPO比DAPO快2x——在什么模型规模？什么任务？是端到端还是单步？"
- "你把训推一致性放在横切约束——但ProRL的token-in/token-out方案不就是把它工程化解决了吗？这还算未解挑战？"
